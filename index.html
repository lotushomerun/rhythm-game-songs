Rhythm Game Songs!

On this website you'll find songs for rhythm game mappers to choose to map!

This website is sorted into genres, including Electronic, Vocaloid, and Anime!

Original metadata is shown, no romantizations except if the title sounds like english but is in kana.

All audio files are 320kbps vorbis (.ogg) or the best quality I can find so you can directly use them to chart
